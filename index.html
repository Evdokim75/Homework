Головна Галерея
  1. git status - перегляд змін
  2. git add - показує відмінності щодо внесених змін у ще не проіндексованих файлах
  3. git branch - іменовані серії коммітів та поєднання результатів роботи
  4. git log - перегляд та вивчення історії змін файлів проекту
  5. git push - вносить зміни з гілки віддаленого репозиторію в поточну гілку локального репозиторію
  6. git commit - робить для проекту знімок поточного стану змін, доданих до розділу проіндексованих файлів
